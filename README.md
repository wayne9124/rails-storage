# 環境

* Ruby 2.3.4

* Ruby 5.2.0

* System MacOS

* Database mysql

* Services Redis(5.2以後的很簡單(撒花))

# 主要測試之技術

- Active Storage (含 `mini_magick`)

- MQTT

- 行事曆(`gem 'fullcalendar-rails'`, `gem 'momentjs-rails'`)，移動行程可更新(AJAX)

- Service's controller 有呼叫 Job 的 ItemJob(測試抓法拍屋資料)(Nokogiri練習)(regex練習)
